Colorfy Color Blind Test

ColorFy utilizes a color vision test known as an Ishihara Test to determine if you have the possible traits resulting in red-green color blindness. You will be provided 14 color plates alternating in color patterns and size dots within the picture. This quiz is intended for entertainment purposes only and does not suffice as a medical diagnostic.


Instructions: Select the start button to begin the quiz. Upon beginning the quiz type the number you see indicated in the circle. If no number is visible do not type and answer and select next. Upon completing all 14 questions you will be provided with your results.
